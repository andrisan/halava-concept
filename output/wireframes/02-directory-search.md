## Overview

The Directory & Search feature allows consumers to discover halal places (restaurants, grocery stores, services) with map and list views. Key screens include search, results, filters, and detailed place pages with halal certification information.

## Screens

| Screen | Path | Mobile | Desktop |
|--------|------|--------|---------|
| Search Bar | `/search` | Full-screen | Topbar |
| Search Results | `/search?q=...` | Full-screen list | Split view (list + map) |
| Map View | `/places` | Full-screen map | Integrated in split view |
| Filter Panel | `/search` | Bottom sheet | Sidebar |
| Place Page | `/places/{slug}` | Full-screen page | Modal or side panel |
| Halal Certificate | `/places/{slug}` | Full-screen page | Modal |

## Implementation Notes

### Key Components Needed

- `SearchBar` - Input with autocomplete, recent/popular suggestions
- `SearchResults` - Tab bar (Places/Products), result list
- `PlaceCard` - Image, name, rating, cuisine, distance, status
- `MapView` - Interactive map with markers
- `FilterSheet` - Bottom sheet on mobile, sidebar on desktop
- `PlacePage` - Full place details with tabs
- `PhotoGallery` - Swipeable image carousel
- `HalalCertificateModal` - Certificate display with disclaimer
- `AmenityChip` - Small badge for amenities

### API Dependencies

- `GET /api/search?q=...&type=places` - Search places
- `GET /api/places/nearby` - Nearby places with location
- `GET /api/places/{slug}` - Place details
- `GET /api/places/{slug}/reviews` - Place reviews
- `GET /api/places/{slug}/photos` - Place photos

### Design Tokens Required

- Colors: Open (green), Closed (red), Certified badge
- Typography: Place name, rating, distance, status
- Map: Marker styles, cluster styles
- Shadows: Card elevation, modal overlay
