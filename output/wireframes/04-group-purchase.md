## Overview

Group Purchase allows multiple consumers to coordinate shared orders to reach free shipping thresholds. The initiator creates a group, invites friends via link, everyone adds their items, and the initiator pays the merchant. Participants reimburse the initiator externally (via PayPay, LINE Pay, etc.).

**Core value proposition:**
- Friends/families coordinate bulk orders together
- Reach free shipping thresholds by combining orders
- Even with expensive delivery, cost becomes affordable when split by group

## Implementation Notes

### Key Components Needed

- `CreateGroupForm` - Group name, deadline selection
- `GroupLobby` - Participants list, items, totals
- `ParticipantCard` - User avatar, items, subtotal
- `ShippingProgressBar` - Visual progress to free shipping
- `InviteShareSheet` - Link copy, social share buttons
- `ParticipantBreakdown` - Post-order receipt by person
- `DistributionTracker` (v1.2) - Checklist with map integration
- `ReimbursementRequest` (v1.2) - Payment request notification

### API Dependencies

- `POST /api/groups` - Create new group
- `GET /api/groups/{code}` - Get group details
- `POST /api/groups/{code}/join` - Join group
- `POST /api/groups/{code}/items` - Add items to group
- `DELETE /api/groups/{code}/items/{id}` - Remove item
- `POST /api/groups/{code}/submit` - Submit order (initiator)
- `GET /api/groups/{code}/breakdown` - Get participant breakdown

### Design Tokens Required

- Colors: Initiator highlight, progress bar fill
- Typography: Participant names, amounts
- Progress: Shipping threshold bar
- Icons: User avatars, share icons

### Integration Points

- **Checkout Flow:** Group checkout differs from individual
- **Share APIs:** Native share sheet for LINE, WhatsApp
- **Maps:** Distribution route optimization (v1.2)
- **Payment Apps:** Deep links to PayPay, LINE Pay (v1.2)

### Participant States

| State | Display |
|-------|---------|
| Browsing | No items yet |
| Has Items | Item list with subtotal |
| Ready | Checkmark indicator |

### Group States

| State | Description |
|-------|-------------|
| **Open** | Accepting participants and items |
| **Submitted** | Order placed, awaiting fulfillment |
| **Delivered** | Order received by initiator |
| **Distributed** | All items distributed (v1.2) |
| **Expired** | Deadline passed without submission |
