## Implementation Notes

### Key Components Needed

- `GlobalTopbar` - Logo, search, location, icons
- `BottomTabBar` - Mobile navigation (Home, Search, Cart, Profile)
- `HeroBanner` - Swipeable carousel for promotions
- `QuickCategoryGrid` - Icon + label tiles
- `ProductCard` - Image, name, price, rating, save button
- `RestaurantCard` - Image, name, rating, cuisine, distance, status
- `CategoryTile` - Image + label
- `PromoCard` - Promotional banner with code
- `DiscoverFeed` - Infinite scroll container (mobile)
- `DiscoverRestaurantCard` - Large hero card for restaurants in feed
- `DiscoverProductGrid` - 2-column product grid in feed
- `FeedFilterTabs` - Filter tabs (All, Restaurants, Products, Deals)

### API Dependencies

- `GET /api/products/trending` - Trending products
- `GET /api/places/nearby` - Nearby restaurants (requires location)
- `GET /api/categories` - Category list
- `GET /api/promotions/active` - Active promotions
- `GET /api/feed/discover` - Discovery feed (paginated, infinite scroll)
  - Query params: `?cursor=...&filter=all|restaurants|products|deals`
  - Returns mixed content: restaurants and products
  - Personalized based on location and user preferences

### Design Tokens Required

- Colors: Primary, secondary, success (open), error (closed)
- Typography: Headings, body, caption, price
- Spacing: Card padding, grid gaps, section margins
- Border radius: Cards, buttons, inputs
- Shadows: Card elevation

### Interactions

**Mobile:**
- Swipe hero banner
- Tap product card â†’ Product page (full-screen)
- Tap restaurant card â†’ Place page (full-screen)
- Tap category â†’ Filtered search (full-screen)
- Pull to refresh (top of page)
- Infinite scroll in Discover feed (loads more on scroll)
- Filter tabs in Discover section
- Quick add to cart from product cards in feed
- Bottom tabs navigation

**Desktop:**
- Hover effects on cards (elevation change)
